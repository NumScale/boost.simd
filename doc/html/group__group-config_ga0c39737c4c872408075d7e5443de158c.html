#include <boost/simd/config.hpp>

Denormals handling behavior option.

By defining this symbol, Boost.SIMD mathematical functions will ignore denormals passed as input and won't try to behave correctly if denormals are generated in intermediate computations. This option usually provides increased performance at the cost of correct handling of denormals related corner cases.
